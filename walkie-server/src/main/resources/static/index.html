Edit the name above and click Rename (or press Enter in the field).
Global push-to-talk uses one shared channel for everyone — the name is fixed to global. Switch the Channel mode away from global to join a different channel.
The global channel is the server's unencrypted broadcast room — encryption isn't available there.
Change the transport, mode, channel, passphrase, or owner above, then Apply — or Reset to discard the pending changes. A new channel name switches rooms (carrying the chosen properties); otherwise the change applies to the current channel — mode, passphrase and ownership are owner-only, and changing the transport reconnects. Clear the passphrase and Apply to turn encryption off.
